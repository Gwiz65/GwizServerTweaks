# GwizServerTweaks
Some Wurm Unlimited server tweaks - mostly focused on improving solo gameplay.

- **Installation**
  - needs Ago's Server Modlauncher installed
  - extract gwizservertweaks-x.x.zip into Wurm Unlimited Server directory
 
- **GwizServerTweaks Features**
  - Allow inter-faith priest linking
  - Make spirit guards attack unique creatures
  - Make tower guards attack unique creatures
